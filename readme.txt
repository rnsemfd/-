android studio를 이용해 만든 클라이언트 <-> 서버 <-> python
으로 연동해둠.

클라이언트로 빵의 정보 및 가격으로 결제를 도와주고
서버를 통해 데이터를 주고받으며
파이썬에서 이미지 분류 후 구분을 담당한다.

--------------------------------------------------------------------------------------------------

발표 ppt나 프로젝트 보고서를 이용해 전체적인 코드 및 구상을 볼 수 있다.
여기에는 내가 주로 담당한 딥러닝/머신러닝 파트만 구성해두었다.

---------------------------------------------------------------------------------------------------

make_bread_data 을 실행시 이미지를 데이터화 하면서 갯수를 늘림. 약 5~10분 소모

bread_classify 을 실행시 데이터로 학습을 하는 단계. 기기 성능에 따라 시간 소모됨

bread_classify_run 을 실행시 학습된 데이터로 테스트에 있는 사진들을 예상함. 

이미지 파일을 모으기 위해 만든 크롤링 프로그램으로 두가지가 있다.

flicker의 api를 이용해 만든 flicker의 이미지만 가져오는 프로그램과
이것이 부족하여 추가로 google에서 긁어오는 google용이 있다.
